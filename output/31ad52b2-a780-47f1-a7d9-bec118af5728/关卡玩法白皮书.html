00 / Core Gameplay

核心体验循环

一句话概括：在微缩世界用射击破坏一切、自动采集材料、海克斯构筑与英雄强化越打越强，围绕单塔推塔展开 15 分钟一局的休闲大乱斗。整套玩法由一条自驱动循环串联——射击破坏、边战边采、强化构筑、交战推塔、战局结果，战后探索。

射击破坏→ 自动采集→ 强化构筑→ 交战推塔→ 战局结果→ 战后探索
局内玩法以「采集 → 强化」为主轴：一边战斗一边自动收集材料，再把材料目标导向地转化为海克斯构筑与英雄强化。两者环环相扣，构成「越打越富、越富越强」的局内驱动力。杜绝传统生存游戏「停下脚步专门打工」的负面体验，让采集与清线、刷野、推塔、交战高度融合。

边战边采

射击破坏体素物件、小兵与建筑，掉落资源自动吸附入袋，以战养战，无需停手专门打工。

强化构筑

采集到的材料目标导向地转化为海克斯随机强化与英雄成长，所见即所得地把资源变成战力。

海克斯流派

随等级发放多轮局内随机强化，技能变异与流派构筑无限组合，每局都是全新 Build。

推塔博弈

单塔三命机制 + 劣势补偿 + 三段终结，15 分钟一锤定音，永不坐牢、绝无膀胱局。

玩法框架

把分散的系统收拢成一张可读的蓝图：四层金字塔 × 三条价值主线 × 五类对局模式。自下而上层层供能，依次解决「好上手 → 有深度 → 有策略 → 留得住」——让新玩家 30 秒上手，老玩家 300 局不腻。

一、能力金字塔 · 四层供能1 → 2 → 3 → 4 循序渐进

从最底层的操作手感起步，循序渐进逐层叠加——① 好上手 → ② 有深度 → ③ 有策略 → ④ 留得住。下层为上层供能，上层为下层赋予意义，玩家的成长体验沿这条阶梯层层递进。

资源与采集系统

实现「战斗即采集」的流畅体验，底层按破坏手感区分为四大元素，每类对应一种独特的碎裂质感（软 / 结构 / 硬 / 能量），视觉形态用于丰富场景表现，部分英雄技能与具体元素联动。

🌿 软质 · 有机

植物 / 食物 / 纸。视觉表现：蛋糕、水果、叶子、纸巾等。软碎飘散手感，高频小量产出；治疗 / 暴食型英雄吃软质资源触发额外加成。

🪵 木质 · 结构

积木 / 木料 / 文具。视觉表现：积木、铅笔、火柴、纸盒等。劈裂块碎手感，中量建材产出；结构型英雄破坏木质叠加掩体 / 修复联动。

⚙️ 金属 · 硬质

铁件 / 工具 / 硬币。视觉表现：铁罐、剪刀、回形针、硬币等。弹射火花手感，低频高量硬通货；金属系英雄触发腐蚀 / 溅射联动。

💡 能量 · 电子

电池 / 芯片 / LED。视觉表现：电池、电容、遥控器、LED 等。爆炸电弧手感，稀有能量爆点；电子系英雄触发连锁电爆 / 探测联动。

⟳
自动化

射击即吸附

破坏 / 击杀后掉落资源自动吸附到玩家身上，无需弯腰拾取。

获取途径：场景体素破坏、击杀小兵、摧毁建筑、击杀敌方英雄——除常规金币经验外，均额外爆出对应资源。采集行为与清线、刷野、推塔、交战高度融合。
⚡ 统一能量体结算：软质·有机、木质·结构、金属·硬质、能量·电子四类元素资源在被击破后统一转化为「能量体」给予角色使用——底层只是同一种可吸附的能量货币，元素分类仅用于差异化破坏手感、丰富场景表现与少数英雄技能联动（如治疗型吃「软质」加成），不增加采集与结算的理解成本。

02+ / Growth

成长体验

玩家成长由基础战斗 + 技能 + 海克斯三层承接（对齐《3C 战斗白皮书》）：前两层由角色设计奠定差异化底子与流派主轴，海克斯则在局内通过随机强化无限演化 Build

角色设计Hero

英雄是承载战斗玩法的核心载体。在统一的战斗基石之上，每个英雄围绕「弹道即技能 + 体素破坏联动被动 + 材质采集流派」三位一体设计，让「射击—破坏—采集—强化」核心循环呈现差异化的战斗节奏与构筑路线。

战斗基石（对齐《3C 战斗白皮书》）：越肩近景 TPS + PvEvP 多方对局 + 全员远程 + 1 角色 1 枪（普攻 A）+ 1 通用技能 + 1 体素破坏技能 + 强档辅助瞄准（非硬锁）。枪是技能释放器，技能与体素破坏资源深度挂钩，角色差异化由基础战斗 + 技能 + 海克斯三层承接。

越肩近景 TPS · PvEvP

第三人称越肩近景、轻俯角跟随；支持 PvE（环境破坏与采集）与 PvP（英雄交战与推塔）同步进行，体素世界是既是战场也是资源库。

全员远程 · 1 角色 1 枪

首批 8 个角色，1 角色绑定 1 把专属枪、不做换枪；全员远程、不出近战，靠枪型与技能正交组合拉开差异。

技能设计模板

每角色固定：普攻 A（枪）+ 技能①通用（位移/辅助/进攻）+ 技能②破坏（与体素 / 材料挂钩），不设大招键，输入极简。

强档辅助瞄准 · 非硬锁

子弹微弯 + 准星吸附消除微操焦虑，但绝不硬锁，朝向永远由玩家控制——移动端 / 手柄 / 键鼠都友好。

体素破坏 · 技能联动

技能②与体素物件直接联动：命中体素触发额外效果、分层爆破产出资源、建筑注针获得补给——破坏是技能的核心输入。

三段式命中

受击碰撞体三段式：头 ×1.2 / 身 ×1.0 / 腿 ×0.8，差距留窄、爆头给独立 VFX / SFX，爽感拉满又不逼精瞄。

武器手感 · 弹道即技能（四个原型角色）

以四个角色原型为弹道代表，在「伤害 / 射速 / 破坏力 / 射程」四维之外，本作额外强化破坏维度与技能联动维度——武器弹道与技能效果深度咬合，让枪型本身具备辨识度。

英雄原型库 · 四大流派范本

首批共 8 个角色（此处示意 4 个原型），覆盖战斗 / 辅助 / 探索 / 拆塔四条流派主轴。每个英雄严格对齐《3C 战斗白皮书》的统一框架——「1 角色 1 枪（普攻 A）+ 1 弹道技能 + 1 位移技能、不做大招」，弹道技能取自弹道池、位移技能取自位移池，两池正交组合；再叠加与「体素破坏 + 材质采集」深度绑定的被动，形成差异化的战斗节奏与构筑路线。

角色设计模板（对齐《3C 战斗白皮书》）：1 把专属枪（普通攻击 A，弹道形态即枪型差异）+ 2 个技能键，不设大招——① 弹道技能（直射 / 散射 / 追踪 / 抛物 / 范围 / 穿透 / 弹射 / 充能 / 蓄力分裂）与② 位移技能（冲刺 / 翻滚 / 闪现 / 二段跳 / 钩锁 / 隐身位移）正交组合。本作在此基础上让两个技能的效果与体素破坏 / 材质采集深度联动（即「破坏 / 资源」表现），并搭配「可拓展效果」供海克斯变异解锁。

03+ / Destruct

可破坏物体系统

「破坏」是本作关卡玩法的底层引擎——它不只是视觉爽点，更是串联资源产出、战斗驱动、地形改变与战术博弈的核心闭环：射击破坏场景物件 → 爆出战斗资源 → 驱动战斗节奏 → 影响战术格局 → 改变关卡地形，循环往复。

核心价值闭环

破坏场景物件→ 获得战斗资源→ 驱动战斗→ 影响战术→ 改变关卡地形↺

分类体系

双维度划分：按材质元素（🌿软质·有机 / 🪵木质·结构 / ⚙️金属·硬质 / 💡能量·电子，决定破坏手感与碎片表现）与按物体本身（资源箱 / 掩体 / 地形 / 装饰）区分功能定位。

产出内容

破坏结算掉落 经验 · 金币 · 材料 三类核心收益；产出受物件类型、品质与破坏条件影响，并设单局上限防止无限滚雪球。

场景占比

地图中预设一定比例物件为可破坏，平衡「可交互感」与「性能 / 地形可控性」，关键掩体与地形破坏点经过设计编排。

随时间演化

产出数值、破坏密度等关键参数随对局时间动态上调，呼应 15 分钟节奏曲线，让中后期破坏收益与场面强度持续升级。

破坏机制 · 多阶段血量模型

采用血量机制，破坏分阶段推进——每阶段触发外观变化与对应事件（掉落碎片、阶段性产出、地形塌陷）。可叠加物体专属 Buff（如短时无敌 / 受击免疫）来控制破坏节奏，避免被瞬间秒破。

Stage 1 · 完好
初始形态

满血，作为掩体 / 装饰，外观完整，提供视野与枪线遮挡。

Stage 2 · 受损
外观裂变

掉血触发裂纹 / 缺角等外观变化，掉落少量碎片，给予清晰破坏反馈。

Stage 3 · 破碎
事件触发

临界阶段触发特殊事件（爆开、塌陷、连锁），可设短时无敌避免瞬秒。

Stage 4 · 摧毁
结算产出

完全破坏，结算并自动吸附经验 / 金币 / 材料，留出通路或改变地形。

战斗交互 · 尺寸分级

可破坏物体按尺寸分级，直接影响视野、枪线与走位——半人高物件提供蹲伏掩护，一人高阻挡站立枪线，巨型物件甚至可临时遮挡防御塔，创造「拆掩体开视野」的攻防博弈空间。

半人高

蹲伏掩体，可越顶射击；破坏后开辟低位枪线。

一人高

完全阻挡站立枪线与视野，是争夺 / 拆除的关键掩体。

巨型 / 遮塔

体量足以遮挡防御塔，改变进攻视野，破坏即战略级开局。

性能方案 · 大规模破坏的工程保障

设计侧控规模 × 开发侧控开销

01

同屏规模管控

预设同屏可破坏物体数量、单次射击可破坏上限、破坏后碎片 / 粒子的同屏峰值，从设计侧约束规模。

02

服务器校算边界

明确哪些破坏行为需后台服务器校算（关键资源 / 地形），哪些纯表现端处理，平衡公平性与负载。

03

碎片池系统

采用对象池复用破坏碎片，避免高频实例化 / 销毁带来的 GC 抖动与卡顿。

04

LOD 分级

按距离 / 重要度对可破坏物体与碎片做细节分级，远处简化表现，保障帧率稳定。

待解决的关键设计问题

四元素材质（软质·有机 / 木质·结构 / 金属·硬质 / 能量·电子）的破坏手感如何与碎片表现精确映射？是否需要混合材质物件？
每种资源在单局内的产出上限与成长上限如何设定？
破坏的各血量阶段分别触发哪些具体事件？是否都需要差异化外观？

服务进程 ·劣势方发育补偿

传统大乱斗给劣势方的核心情绪是「缩塔下清线就能苟住等翻盘」。我们因「出塔采野区资源才能强化」打破了该逻辑，故从四个维度补偿，让劣势方掉塔后依然有发育空间组织反扑。

① 防御塔形态升级

扩大安全区与威慑力，掉命即「暴走/过载」。

  • 射程与真实视野 +20%（掉二命 +30%），把危险区划入安全区
  • 攻击质变 AOE 溅射 / 多目标扫射，毁灭压境兵线与越塔单位
  • 掉命瞬间 30 秒「绝对防御」护盾，打断敌方连续推进

② 塔下安全资源区

解决「断粮」危机，把资源送到家里。

  • 掉命瞬间塔后安全区刷大量高品质补给箱，开枪击碎即得
  • 足够立刻拉满一轮海克斯强化或补满护盾 / 弹药
  • 防守反击资源翻倍（Salvage）：塔射程内击杀敌方掉落翻倍 2-3 分钟

③ 团队反扑 Buff

提供冲锋与拉扯资本，敢于重回中心。

  • 「绝地反击」光环：出塔向中心移动获高额移速 + 防弹玩具护盾（抵一次致命控制/爆发）
  • 海克斯「败者补偿」：经济差距过大时，下一轮必出高稀有度/强力翻盘选项

④ 兵线与建筑减伤（防猝死底线）

借鉴传统 MOBA 优秀经验 + 体素特色。

  • 防偷塔：射程内无敌方兵线时，塔受伤 -80%，必须带线才能推
  • 体素修复：满状态吃到木质·结构类资源，多余建材飞向塔缓慢修复（微量回血）

07 / Closer

服务进程 ·优势方终结手段

为打破劣势方「无限缩塔清线」导致的膀胱局，给优势方一锤定音的终结手段。三个机制递进：10 分钟兵线变强 → 12 分钟盲盒逼团 → 15 分钟水晶大碰撞。

10 分钟 · 攻城兵线狂暴

随时间成长的破防能力，防止劣势方 AOE 轻松清掉进攻兵线。

  • 10 分钟后所有进攻方兵线自动获【攻城狂暴】词条
  • 体型更大、血量更高，攻塔无视部分减伤
  • 时间越往后，无视减伤比例越高
  • 持续控野区滚海克斯雪球，劣势方防线迟早被撕裂

12 分钟 · 奇迹盲盒

常规对局最核心终结手段，高价值目标逼劣势方出塔接团。

  • 正中心天降【奇迹盲盒/发条宝箱】
  • 需长时间读条/拧发条，受击暂停——强迫劣势方出塔打断
  • 开启爆海量资源，全队瞬间拉满海克斯强化
  • 全队「过载」Buff：体型变大、移速增加、建筑伤害提升

15 分钟 · 玩具盒大暴走

异常焦灼拖到 15 分钟，强制「突然死亡」彻底终结。

  • 双方防御塔长出机械腿/履带拔地而起走向中心
  • 塔脱离安全区在中心相遇互殴（极限闪击式碰撞）
  • 无兵线高额减伤彻底失效
  • 谁的塔先被打掉最后一条命谁输

一句话回写

关卡玩法核心方向：体素破坏 + 战斗即采集 + 海克斯无限构筑 + 单塔三命。地图为单点冲突的中心交战区兼资源区，2-3 层立体纵深。防御塔三命掉命触发扩区/爆资源/团队 Buff四维补偿，让劣势方「掉命即反扑」而非坐牢；优势方靠 10 分钟兵线狂暴 → 12 分钟奇迹盲盒逼团 → 15 分钟水晶大碰撞三段递进一锤定音。双轴节奏（海克斯 6 轮 + 地图事件）把单局严格压在 15 分钟。

07+ / Extraction

体素撤离玩法

在主玩法「对抗推塔」之外，开辟一条高风险高回报的 PvEvP「搜打撤」支线（对标《暗区突围》《逃离塔科夫》，融合本作「体素破坏即采集」内核）：玩家携带装备进入微缩战场，「搜」——破坏搜刮积累战利品；「打」——在 AI 与他队的压力下交火博弈；「撤」——抵达撤离点读条带出收益。撤离成功则收益入库永久保留，中途阵亡则背包全数掉落，把「贪 or 走」的张力拉满。

为什么适配本作：本作「射击破坏一切体素 → 资源自动吸附」的核心循环，天然就是最爽快的「搜刮」动作——搜刮即破坏、破坏即采集，无需翻箱倒柜的繁琐交互，把撤离品类最枯燥的搜包环节，变成本作最解压的破坏快感。

一、风险—收益曲线Risk × Reward

越往后期，收益越高、风险也越大。系统通过时间轴持续加压，迫使玩家在「见好就撤」与「贪一波大的」之间做出抉择——这正是撤离品类的核心心流。

0–5 min · 安全期
外围搜刮

地图外圈低风险破坏搜刮，积累基础元素资源与普通战利品；遭遇少量弱 AI，适合稳健起步。

5–10 min · 收缩期
毒雾推进

边缘毒雾开始收缩，安全区缩小；高价值区域（电子机房 / 金属保险库）开放，玩家相遇概率上升。

10–14 min · 高压期
精英与空投

AI 精英 Boss 刷新、稀有空投降临中心，必出史诗 / 传说战利品；撤离点陆续激活，团战频发。

14–18 min · 终局
最后撤离窗口

仅剩 1 个撤离点开放且读条时间翻倍，毒雾逼至核心；不撤离即淘汰，逼出终极抢点对决。

二、撤离点状态机Extraction Point

地图分布多个撤离点，随时间轮换激活状态。撤离需站桩读条，读条期间暴露身位且可被打断，制造「最后一公里」的紧张博弈。

🟢
已激活 · 可撤离

站点读条 8 秒即可撤离，受击 / 离开范围中断，需队友架枪护点。

🟡

显示激活倒计时，提前蹲守占位，但易成为他队伏击的钓鱼点。

🔴
热点 · 终极点

终局唯一开放点，读条时间翻倍、全图可见信标，必有抢点死斗。
