调律者 世界观设定 近未来都市 · 心理悬疑 · 超能力竞技

当所有人的痛苦都需要由少数人来净化时，那些负责守护光的人，又该由谁来守护？——这是一个用少数少年少女净化集体情绪污染、并追问这套系统本身是否正义的世界。

近未来都市 心理悬疑 超能力竞技 情绪净化 跨媒体可拓展
01 / Era

时代背景 ·被情绪侵蚀的近未来都市

高速运转的近未来社会让负面情绪在隐秘维度「亚相层」中沉积，溢出为名为「晦能」的负性力量，悄无声息地污染城市与人心。

在近未来的现代都市中，社会长期高速运转所带来的副作用逐渐浮现：焦虑、内卷、孤独、愤怒与无力感，已不再只是个人心理问题，而演化为一种足以改变现实的集体性污染。

人类的负面情绪持续在现实背后的隐秘维度——「亚相层」中沉积。这个原本只用来映照心灵波动的空间发生失衡，开始通过时空中的裂隙，向现实世界反向溢出一种名为「晦能」的负性力量。

城市的异常征兆

晦能并不会凭空创造灾厄，它只做一件事：精准放大人们心中已有的疲惫、嫉妒、怨恨、恐惧与自我否定。最初的征兆细微而日常：

  • 身边的人越来越容易发火
  • 学生厌学比例急剧上升
  • 网络争吵变得更加极端
  • 城市某些区域让人莫名压抑

污染等级

程度表现
轻度麻木、颓废、易怒、抑郁
中度持续性精神低气压，失去正向动力
重度精神崩坏，部分区域形成小规模情绪灾害与污染区域
直到污染区域开始在城市中可视化地扩散，人类才后知后觉地发现：原来一切的源头，都指向那个被忽视已久的隐秘维度——亚相层。
02 / Setting

核心设定 ·亚相层与晦能

亚相层是中性的心灵投影空间，被集体负面情绪持续污染后失衡，向现实溢出「晦能」。失控聚集时形成「心魇域」——会同步周围人情绪的灾厄领域。

亚相层维度概念

Sub-Phase Layer

现实背后的隐秘维度，原本是中性的「心灵投影空间」。

  • 本身不善不恶，只是被动映照集体心灵波动
  • 当人类整体情绪长期偏向负面，亚相层就被持续污染
  • 失衡后，它开始反向向现实溢出晦能

晦能核心威胁

Hai-Energy / 负性能量

由集体负面情绪沉积、压缩、变质而成的非物质性力量。

  • 不会凭空制造灾厄
  • 放大已存在的负面情绪
  • 长期成年人是它最稳定的寄主（精神结构僵硬）
  • 在情绪激烈处汇聚，在城市裂隙处溢出

心魇域灾厄领域

Mind-Nightmare Zone

当晦能在某个空间或某个个体身上失控聚集时，所形成的污染领域。

  • 领域内：周围人的情绪会被强行同步、扭曲并拖入失控循环
  • 是「魇化者」的天然战场和能量来源
  • 城市级别的心魇域 = 大规模情绪灾害

晦能的恶性循环

情绪低气压 → 行为消极 → 释放更多负面情绪 → 加剧晦能浓度
     ↑                                              ↓
     ←────────────── 闭环侵蚀 ─────────────────────
03 / People

关键人群 ·调律者与魇化者

少年少女们能吸收并净化晦能成为「调律者」，能力源自他们最深的执念。但承载有上限，超过临界点会异化为「魇化者」——曾经的守护者，如今的灾厄。

3.1 调律者 / Tuner

能够吸收并净化晦能的特殊个体。他们不是救世主，只是被命运推上前线的少年少女。

共同特征

  • 不会被晦能直接影响（具有天然抗性）
  • 能像过滤器一样吸收周围溢散的负面能量
  • 通过净化重组，将晦能转化为可控的「澄能」
  • 从而觉醒出具象化的超能力

为什么大多是青少年？

群体精神结构与晦能的关系
成年人受社会规则与生存压力塑形，稳定但僵硬晦能天然的寄生温床
青少年精神结构尚未完全固化，保留强烈「可能性」拥有净化能力的根基

能力的本质：信念的具象化

每位调律者的能力都与其内心深处最强烈的执念、渴求或愿望相连：

想保护某个特定的人

想证明自己不是无价值的

想揭开某个真相

想把世界变得稍微好一点

正是这份信念，让他们能将混乱的晦能重塑为有形的力量。

已观测到的能力类型

PHYSICAL · 物理干涉

操纵重力

改变局部重力场方向、强度或矢量。

PHYSICAL · 物理干涉

具现化武装

将精神图景投影为可使用的武器实体。

SPATIAL · 空间干涉

空间扭曲 / 瞬移

短距空间折叠、领域展开。

PERCEPTUAL · 感知干涉

声音操纵

声波武器化、共振破坏、定向收音。

PERCEPTUAL · 感知干涉

影子干涉

操纵自身或他人影子，产生独立行为。

MNEMONIC · 记忆干涉

记忆残响

读取或重现物体/空间留存的记忆碎片。

能力类型随调律者执念的不同而无限延伸，没有两个完全相同的调律者。

3.2 魇化者 / Nightmare-Bound

调律者失控后的灾厄形态。每一个魇化者，曾经都是想保护他人的少年少女。

形成机制

吸收晦能 → 持续承压 → 情绪杂质反向侵蚀 → 超过临界点
                                                ↓
                              净化失败的晦能体内逆流
                                                ↓
                              记忆与人格开始扭曲
                                                ↓
              ★ 异化为「魇化者」
              由破碎心愿 + 负面执念 + 超能力残骸构成的灾厄存在

危害

  • 会在现实中展开「心魇域」污染领域
  • 把周围人的情绪一同拖入失控循环
  • 是城市灾害的最高威胁等级
悲剧性核心：当一个调律者最深的信念无法承担时，他想守护的、想证明的、想揭开的、想改变的——全都会反噬他自己。
04 / System

能力体系 ·晦能 ⇌ 澄能

调律者吸收晦能后，以信念为催化剂将其净化重组为「澄能」，具象化为超能力。但承载量遵循 5 阶段风险曲线：从安全 → 警戒 → 高压 → 临界 → 异化。

4.1 转换公式

        ┌──────────────┐
        │    晦能      │  ← 集体负面情绪沉积，混乱、有毒
        └──────┬───────┘
               │ 调律者吸收
               ▼
        ┌──────────────┐
        │  净化重组    │  ← 信念作为催化剂
        └──────┬───────┘
               │
               ▼
        ┌──────────────┐
        │    澄能      │  → 具象化为超能力
        └──────────────┘

4.2 承载上限与风险曲线

调律者并非完美容器，吸收越多，越接近异化临界点。

05 / Arena

社会系统 ·共鸣校准场

表面是华丽的超能力赛事，本质是融合心理治疗、防灾、能源转化的复合设施。每一场竞技 = 一次城市级情绪泄洪 + 心核透析手术。

5.1 表象与本质

维度对外表现真实本质
视觉超能力竞技场心理治疗 + 防灾工程 + 能源转化 + 战斗训练 的复合设施
体验华丽的超能力对抗赛城市级情绪泄洪 + 心核稳定手术 + 灾害预演
商业高规格的明星赛事调律者集中净化晦能的「透析中心」

5.2 内部运行机制

校准场内部，空间被规则结界封闭，形成一个完全可控的高压安全环境：

  1. 激发 · 调律者通过受控对抗激发情绪峰值
  2. 释放 · 体内积压的晦能在高压安全环境中被排出
  3. 碰撞 · 通过队友/对手能力碰撞完成「心核频率校准」
  4. 回收 · 所有攻击、损伤、情绪波动都被系统记录、削减并转化
  5. 净化 · 避免能量长期淤积造成逆流

5.3 为什么是「竞技」？

晦能源自冲突、压抑、失衡。只有在明确规则、清晰胜负、可控压力、稳定人际连接中被重新激活，才能从混乱的情绪污泥，转化为可使用的澄能。竞技不是娱乐，而是最有效的心核透析流程。

5.4 公开化与产业化

随着晦能灾害常态化，共鸣校准场从秘密医疗设施扩展为城市基础系统：

内部用途（非公开）

  • 新晋调律者的训练基地
  • 城市突发情绪灾害的应急救援中心
  • 调律者的心理治疗与生命维持
  • 心魇域爆发的预演与战术演练

对外赛事（高度商业化）

  • 明星调律者参赛，万人观赛
  • 观众情绪 → 形成定向晦能源
  • 调律者集中吸收 → 通过竞技完成转化
  • 一场比赛 = 一次城市级情绪泄洪
关键洞察：观众的聚集本身就是「情绪源汇」。看似在看比赛，其实在被定向收集情绪。但这并不邪恶——这是一种双赢的设计：观众宣泄了情绪，调律者完成了净化，城市消除了灾害隐患。
06 / Tier

等级机制 ·调律者竞技等级

从初阶训练生到超阶特殊存在，五个等级对应不同的承载能力与风险。聚光灯越亮，深渊越近——这是调律者的宿命阶梯。

赛事等级越高 → 所要求的调律者承载能力越大 → 净化效率越高 → 风险也越高。
等级含义承载能力风险等级
初阶训练生 / 见习调律者低较低，需导师陪同
中阶注册调律者中可独立执行小规模净化
高阶公开赛事选手高需稳定团队 + 心理顾问
顶阶城市级 / 区域级「明星调律者」极高长期临界压力
超阶极少数特殊存在接近异化阈值随时可能成为魇化者
站在聚光灯下的少年少女们
等级越高，离深渊越近
07 / Drama

戏剧冲突 ·救赎的悖论

华丽表象与残酷真实之间的巨大落差，构成了世界观的戏剧张力。「守护者由谁守护」是贯穿全篇的核心追问。

7.1 表象 vs 真实

表象

  • 华丽的超能力赛事
  • 万众瞩目的明星
  • 胜负 / 荣誉 / 商业化
  • 守护他人的英雄

真实

  • 城市级情绪泄洪
  • 替无数普通人承受负能量的危险容器
  • 一场关于自我边界的试炼
  • 随时可能崩坏的少年少女

7.2 核心母题

当所有人的痛苦都需要由少数人来净化时
那些负责守护光的人，又该由谁来守护？

这是贯穿整个世界观的核心追问，也是每一位调律者必须面对的命题：

  • 他们既是被社会寄予希望的明星
  • 也是替无数普通人承受负面情绪的危险容器
  • 在胜负、荣誉、商业化和救赎的背后
  • 始终潜伏着 ——「守护者由谁守护」的残酷悖论

7.3 可衍生的故事母题

成长

从拒绝吸收 → 接受承担 → 学会与负能量共处。

羁绊

搭档调律者之间相互校准心核频率，从陌生到生死相托。

悲剧

当队友魇化，是净化、收容，还是终结？

阴谋

商业化背后是否有人故意制造晦能灾害以提升赛事热度？

自省

调律者自己的负面情绪，又由谁来净化？

体制

当孩子们成为城市的情绪过滤器，这是必要之恶还是结构性剥削？

本作的差异化定位

参考 ≠ 致敬，更不是抄袭。「调律者」在以下维度做了独特设计，与上述作品形成差异：

维度常见作品的处理★ 调律者的独特设计 ★
能量来源外部入侵 / 与生俱来 / 修炼获得来自集体情绪污染，是社会病的副产物
能力机制对抗 / 消灭威胁吸收 + 净化 + 转化——调律者本身就是过滤器
战斗目的击败敌人 / 保护世界用竞技作为城市级「情绪透析」——比赛即治疗
主角群体少数被选中的英雄系统化的「少年容器」群体，本质是替成年社会承压
戏剧张力正邪对决 / 拯救世界守护者由谁守护——结构性悖论而非个体困境
异化命运变强 / 黑化 / 牺牲魇化者就是失败的调律者——救赎对象与被救赎者同体
核心差异化：本作不是「超能力少年打怪救世界」，而是「用少数人的痛苦净化集体情绪污染，并追问这个系统本身是否正义」。这是社会派 + 心理派的少年题材，介于《心理测量者》的冷峻与《我的英雄学院》的热血之间。
