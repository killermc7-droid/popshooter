3C Combat Whitepaper · draft-4 / v0.1

微缩大乱斗 3C 战斗白皮书 越肩近景 TPS · 1 角色 1 枪 · 弹道即技能 · 强档辅助瞄准

枪是技能释放器，不是真实枪械射击模拟器。角色（Character）、相机（Camera）、操控（Control）三大基石，叠加命中判定与技能模板池——角色差异化在中后期由 基础战斗 + 技能 + 海克斯技能 三层承接。

越肩近景 TPS 全员远程 1 角色 1 枪 弹道即技能 强档辅助瞄准 · 非硬锁
00 / Core Pitch

核心定调

越肩近景 TPS + 全员远程 + 1 角色 1 枪 + 弹道即技能 + 强档辅助瞄准（非硬锁）——枪是技能释放器，不是真实枪械射击模拟器；角色差异化在中后期由 基础战斗 + 技能 + 海克斯技能 三层承接。

手感不焦虑

强档辅助瞄准消除微操焦虑，移动端 / 手柄 / 键鼠都友好——一键开火即可打出技能级反馈。

枪即技能

弹道形态就是技能形态。一把枪首发就要做出"小生态"，给后续技能 / 海克斯留接入点。

三层成长

不靠换枪堆深度，靠基础战斗 + 技能 + 海克斯技能三层加深一把枪的玩法。

01 / Character

角色 ·1 角色 1 枪

1 角色 = 1 把专属枪，强绑定、不做换枪系统。全员远程，首批不出近战。武器是技能型枪械——弹道形态即技能形态。角色屏占比偏大，强调辨识度与卡通可爱。

维度决策
角色 ↔ 武器1 角色 = 1 把专属枪，强绑定，不做换枪系统
首批角色数量8 个，分别走不同的"弹道 + 位移"正交组合，保证开局阵容差异化
攻击方式全员远程，首批不出近战
武器定位技能型枪械：弹道形态 = 技能形态（直射 / 散射 / 追踪 / 抛物 / 范围 / 穿透 / 弹射 / 充能 …）
角色体型参考《超自然行动组》——角色在屏幕中占比偏大，强调辨识度与卡通可爱
中长期规划基础战斗 + 技能 + 海克斯技能 三层承接成长 / 多样性（不靠换枪，靠这三层加深一把枪的玩法）

隐含要求 · 每把枪首发就要有"小生态"

既然不能换枪，每把枪首发就要做出基础弹道 + 至少 1 个有辨识度的特性（穿透 / 弹射 / 充能 / 蓄力分裂 …），给后续 技能 / 海克斯技能 留接入点。

隐含要求 · 海克斯最小框架要尽早

海克斯技能（局内随机或局外养成获取的强化）要尽早出最小可玩框架，否则"1 角色 1 枪"长期会被骂内容薄。

02 / Camera

相机 ·越肩近景 TPS

第三人称越肩 TPS，近景、轻微俯角、跟随带缓动。初稿不做精确开镜、不做半瞄拉近反馈、不做肩切换（固定右肩默认）。

维度决策
视角类型第三人称越肩 TPS（角色靠下偏左 / 右，镜头从角色后上方略低俯角看出去）
镜头距离近景——角色占屏高约 1/3 ~ 1/2，肩膀以上清晰可见
俯角轻微俯角（不是平视、也不是俯视）
跟随跟随主角带轻量缓动；移动时镜头略滞后，转向有阻尼
开镜 / 拉近不做精确开镜；初稿阶段也不做半瞄拉近反馈（待后期手感打磨时再评估）
肩切换初稿不做，固定一个肩位（建议右肩默认）

对关卡 / UI 的隐含要求

用纵深做战术

利用前后纵深做战术层次（前排小怪 + 后排远程 + 掩体），而不是平面圆环刷怪。

UI 让出中央

UI 不遮挡角色和准星附近区域；血条、技能、弹药贴四角，中央交给战斗。

避免硬贴墙构图

不做肩切换 → 关卡要避免硬贴墙强制切肩的构图，否则会出现"看不见敌人"的体验事故。

角色模型方向 · 3.5 头身比大头比

采用 3.5 头身比的 Q 版大头身形（参考：扎双辫、戴蓝色兔耳帽、抱着烤面包机的小女孩——卡通可爱、手办质感、强辨识度）。这个比例不是单纯美术取向，而是直接服务于"容易射中 + 容易爆头"的射击手感。

大头 = 大命中盒 → 把"爆头"从精瞄门槛变成人人可享的高频爽点

容易爆头

头部在三段式命中盒里占比天然偏大，配合强档辅助瞄准（落点身体中心 + 准星吸附），爆头从"必须精瞄"变成高频正反馈。

容易射中

短粗敦实的身形目标面积大、轮廓饱满，显著降低脱靶率，呼应"不要求精确瞄准"的核心定调。

辨识度 / 传播

越肩近景下大头身比角色在屏幕里更醒目，表情与装备清晰，卡通可爱、利于社交截图传播。

🔗 与 §4 命中判定协调：3.5 头身比放大了头部命中盒，让爆头"容易发生"；但部位伤害差距仍留窄（头 ×1.2 / 身 ×1.0 / 腿 ×0.8）。即——大头是为了"人人都能轻松打出爆头的爽感与反馈（独立 VFX / SFX 已拍板）"，而不是"逼玩家精瞄才有输出"。"容易爆头"承担爽感反馈，不破坏低操作焦虑的手感方向。
03 / Control

操控 ·强档辅助瞄准

移动 + 开火 + 1~2 技能键的极简输入预算。跳跃是所有角色基础动作；二段跳与冲刺 / 翻滚 / 闪现等保命位移则作为角色技能差异化。辅助瞄准开到强档，但绝不硬锁。

维度决策
移动摇杆 / WASD 八方向
跳跃所有角色基础动作（普攻级能力，不进技能槽）
二段跳特殊技能，不是基础能力——谁有由角色决定
射击一键开火，不需要精确瞄准
辅助瞄准强档：子弹弯曲 + 准星吸附 + 索敌权重全开，但仍不做硬锁定（朝向由玩家给）
半瞄键初稿不做（拉近 / 准星更稳 / 索敌权重提高等反馈暂缓，后期再评估）
保命位移作为角色技能差异化：冲刺 / 翻滚 / 闪现 / 钩锁 / 隐身位移 …，不同角色不同
技能槽枪 = 主输出；预留 1~2 个独立技能键（位移技能为主，不设大招键）
输入预算移动 + 开火 + 1~2 技能键，移动端 / 手柄 / 键鼠都友好

辅助瞄准的边界

强档只消除微操焦虑，不替玩家决定打谁——朝向永远由玩家控制

✓ 强档要做

YES 子弹微微弯向目标（强档幅度可见）

YES 准星接近敌人时吸附（吸附半径较大）

VS

✕ 坚决不做

NO 自动锁头 / 锁敌后 100% 命中

NO 玩家完全不用瞄方向

📐 内部参数（吸附半径 / 子弹弯曲角度 / 索敌优先级权重）需做成可调，便于不同枪型差异化（如狙击型可削弱吸附以保留打中爽感）。

跳跃 / 二段跳的设计含义

跳跃通用

关卡可默认所有角色都能跨小台阶，地形复杂度可适度往上加。

二段跳 = 特权

垂直空间是少数角色的特权，关卡要有"二段跳能上、其他人绕路"的可选路径设计。

可达性标签

关卡编辑器需要"是否需要二段跳"的可达性标签，避免"某角色卡关"。

🎮 跳跃手感基线 → 参考《Splatoon 喷射战士》：单跳高度、滞空时间、落地硬直按 Splatoon 的轻盈灵动手感定基线（跳跃干脆、滞空可控、落地几乎无硬直），与"低操作焦虑 + 卡通爽快"的整体方向一致。后续调参以此为锚点。
04 / Hitbox

命中判定 ·三段式

受击碰撞体三段式：头 / 身体 / 腿，不做更细拆分。各部位伤害差距不大——避免"必须爆头才有输出"，配合强档辅助瞄准与不要求精确瞄准，保持手感顺滑、降低操作焦虑。

×1.2
Head

头

独立 VFX / SFX + 小额加成，主要承担爽感反馈而非数值差异。

×1.0
Body · 基准

身体

基准伤害。子弹弯曲落点统一打到身体中心。

×0.8
Legs

腿

差距仍留窄，不构成精瞄向手感压力。

三段式是数值上限

美术 / 动画做角色时不需要为更精细的命中盒预留骨骼，省工时。差距不大意味着部位伤害更多承担反馈差异而非数值差异。

落点统一身体中心

子弹弯曲（强档辅助瞄准）目标点统一打到身体中心，不为"瞄哪个部位"做策略选择——这与三段式低差距配套。

⚠️ 后期若加"弱点机制"（如某些 BOSS 有红色弱点部位），属于怪物专属逻辑，不污染玩家角色的三段式规则。

待定参数 · 给程序 / 数值的接口契约

参数建议值 / 说明
部位伤害倍率身体 = 1.0 基准，头 1.2，腿 0.8（差距留窄，反馈差异为主）
爆头反馈给独立 VFX / SFX（已拍板）——数值差不大，但爆头爽感拉满
子弹弯曲落点统一目标，默认身体中心
05 / Skill Pool

角色技能模板池

每个角色 = 1 弹道技能 + 1 位移技能，组合即角色个性。不做大招——所有角色只有技能，没有单独的终极爆发键。弹道池决定枪型差异，位移池决定手感差异，两池正交组合，少量条目就能撑出多角色。

1
弹道技能

枪型差异

决定这把枪"打出去什么形态"。

2
位移技能

手感差异

决定这个角色"怎么动 / 怎么保命"。

弹道技能池 · 枪型差异

直射散射追踪抛物范围穿透弹射充能蓄力分裂

位移技能池 · 手感差异

冲刺翻滚闪现二段跳钩锁隐身位移
🧩 正交组合法则：弹道池是枪型差异、位移池是手感差异。两池正交相乘——例如「穿透 × 钩锁」「散射 × 闪现」「追踪 × 隐身位移」——首批 8 个角色即可用少量条目撑出截然不同的玩法，直接支撑高频内容产能。
06 / Benchmark

参考标杆

明确"抄什么 / 不抄什么"，把参考精准锚定在视角、屏占比、关卡构图与弹道手感上，避开精确瞄准 / 开镜射击与负重生存等不相关系统。

🏡 和睦小镇保卫战

抄 越肩近景视角、角色屏占比、卡通色调、纵深关卡构图、UI 布局（左下血条 + 右下技能）

不抄 PvP / 守点玩法节奏（不是 v0.1 重点）

🦑 Splatoon（喷射战士）

抄 高饱和卡通色彩、强辨识度的武器形态差异（不同枪 = 不同玩法）、低门槛爽快射击手感

不抄 墨水涂地 / 变鱿鱼潜行机制、PvP 占领对战模式

📺 瑞奇与叮当

抄 技能弹道手感、释放反馈

不抄 任何精确瞄准 / 开镜射击的部分
