# 大乱斗角色 3C 初稿

---

## 0. 核心定调（一句话）

> **越肩近景 TPS + 全员远程 + 1 角色 1 枪 + 弹道即技能 + 强档辅助瞄准（非硬锁）**——枪是技能释放器，不是真实枪械射击模拟器；角色差异化在中后期由 **基础战斗 + 技能 + 海克斯技能** 三层承接。

---

## 1. Character（角色）

| 维度 | 决策 |
|---|---|
| 角色 ↔ 武器 | **1 角色 = 1 把专属枪**，强绑定，不做换枪系统 |
| 攻击方式 | **全员远程**，首批不出近战 |
| 武器定位 | **技能型枪械**：弹道形态 = 技能形态（直射 / 散射 / 追踪 / 抛物 / 范围 / 穿透 / 弹射 / 充能 …） |
| 角色体型 | 参考《和睦小镇保卫战》——角色在屏幕中**占比偏大**，强调辨识度与卡通可爱 |
| 中长期规划 | **基础战斗 + 技能 + 海克斯技能** 三层承接成长 / 多样性（不靠换枪，靠这三层加深一把枪的玩法） |

### 隐含要求

- 既然不能换枪，**每把枪首发就要做出"小生态"**——基础弹道 + 至少 1 个有辨识度的特性（穿透 / 弹射 / 充能 / 蓄力分裂 …），给后续 **技能 / 海克斯技能** 留接入点。
- **海克斯技能**（局内随机或局外养成获取的强化）要尽早出**最小可玩框架**，否则"1 角色 1 枪"长期会被骂内容薄。

---

## 2. Camera（相机）

| 维度 | 决策 |
|---|---|
| 视角类型 | **第三人称越肩 TPS**（角色靠下偏左 / 右，镜头从角色后上方略低俯角看出去） |
| 镜头距离 | **近景**——角色占屏高约 1/3 ~ 1/2，肩膀以上清晰可见 |
| 俯角 | 轻微俯角（不是平视、也不是俯视） |
| 跟随 | 跟随主角带轻量缓动；移动时镜头略滞后，转向有阻尼 |
| 开镜 / 拉近 | **不做精确开镜**；初稿阶段也**不做半瞄拉近反馈**（待后期手感打磨时再评估） |
| 肩切换 | **初稿不做**，固定一个肩位（建议右肩默认） |

### 对关卡 / UI 的隐含要求

- 利用**前后纵深**做战术层次（前排小怪 + 后排远程 + 掩体），而不是平面圆环刷怪。
- UI 不要遮挡角色和准星附近区域；血条、技能、弹药贴四角，中央交给战斗。
- 不做肩切换 → 关卡设计要**避免硬贴墙强制切肩**的构图，否则会出现"看不见敌人"的体验事故。

## 3. Control（操控）

| 维度 | 决策 |
|---|---|
| 移动 | 摇杆 / WASD 八方向 |
| **跳跃** | **所有角色基础动作**（普攻级能力，不进技能槽） |
| **二段跳** | **特殊技能，不是基础能力**——谁有由角色决定 |
| 射击 | 一键开火，**不需要精确瞄准** |
| **辅助瞄准** | **强档**：子弹弯曲 + 准星吸附 + 索敌权重全开，但**仍不做硬锁定**（朝向由玩家给） |
| 半瞄键 | **初稿不做**（拉近 / 准星更稳 / 索敌权重提高等反馈暂缓，后期再评估是否需要） |
| **保命位移** | **作为角色技能差异化**：冲刺 / 翻滚 / 闪现 / 钩锁 / 隐身位移 …，不同角色不同 |
| 技能槽 | 枪 = 主输出；预留 1~2 个独立技能键（位移 / 大招） |
| 输入预算 | 移动 + 开火 + 1~2 技能键，移动端 / 手柄 / 键鼠都友好 |

### 辅助瞄准的边界

| 行为 | 是 | 否 |
|---|---|---|
| 子弹微微弯向目标 | ✅ 强档幅度可见 | — |
| 准星接近敌人时吸附 | ✅ 吸附半径较大 | — |
| 自动锁头 / 锁敌后 100% 命中 | — | ❌ 不做 |
| 玩家完全不用瞄方向 | — | ❌ 不做 |

> 即使是强档，**朝向仍由玩家控制**。强档只消除微操焦虑，不替玩家决定打谁。
> 内部参数（吸附半径 / 子弹弯曲角度 / 索敌优先级权重）需做成**可调**，便于不同枪型差异化（如狙击型可以削弱吸附以保留打中爽感）。

### 跳跃 / 二段跳的设计含义

- 跳跃通用 → 关卡可默认所有角色都能跨小台阶，地形复杂度可适度往上加。
- 二段跳作为技能 → **垂直空间是少数角色的特权**，关卡里要有"二段跳能上、其他人绕路"的可选路径设计。
- 关卡编辑器需要"是否需要二段跳"的可达性标签，避免"某角色卡关"。

## 5. 命中判定（Hitbox）

| 维度 | 决策 |
|---|---|
| 受击碰撞体划分 | **三段式：头 / 身体 / 腿**——不做更细的拆分（不分手臂、胸腹、关节等） |
| 各部位伤害差距 | **差距不大**——避免"必须爆头才有输出"的精瞄向手感 |
| 设计意图 | 配合"强档辅助瞄准 + 不要求精确瞄准"，保持手感顺滑、降低操作焦虑 |

### 隐含要求

- **三段式是数值上限**——美术 / 动画做角色时不需要为更精细的命中盒预留骨骼，省工时。
- **差距不大**意味着部位伤害更多承担**反馈差异**而非**数值差异**：例如爆头给独立特效 / 音效 / 小额加成，但不构成海克斯必备项。
- 子弹弯曲（强档辅助瞄准）目标点统一打到 **身体中心**，不需要为"瞄哪个部位"做策略选择——这一条与三段式低差距是配套的。
- 后期如果要加"弱点机制"（如某些 BOSS 有红色弱点部位），属于**怪物专属**逻辑，不污染玩家角色的三段式规则。

### 待定参数（给程序 / 数值的接口契约）

- 头 / 身体 / 腿的伤害倍率（建议范围 `身体 = 1.0` 基准，头 `1.1 ~ 1.2`，腿 `0.9 ~ 1.0`，差距留窄）
- 爆头是否给独立 VFX / SFX 反馈
- 子弹弯曲落点的统一目标（默认身体中心）

## 6. 角色技能模板池（建议结构）

每个角色 = **1 弹道技能 + 1 位移技能 + (可选 1 大招)**，组合即角色个性。

### 6.1 弹道技能池

直射 / 散射 / 追踪 / 抛物 / 范围 / 穿透 / 弹射 / 充能 / 蓄力分裂 …

### 6.2 位移技能池

冲刺 / 翻滚 / 闪现 / 二段跳 / 钩锁 / 隐身位移 …

> 弹道池是枪型差异；位移池是手感差异。两池正交组合，少量条目就能撑出多角色。

## 8. 一句话回写

> 大乱斗角色 3C 初稿方向：**越肩近景 TPS + 全员远程 + 1 角色 1 枪 + 弹道即技能 + 强档辅助瞄准（非硬锁）+ 不做肩切换**。所有角色默认带跳跃，**二段跳和冲刺 / 翻滚 / 闪现等位移作为角色技能差异化**。**受击碰撞体三段式（头 / 身体 / 腿），各部位伤害差距不大**，与"不要求精确瞄准"的手感方向一致。视角参考《和睦小镇》，体型偏大、轻俯角、近景。换枪不做，深度交给后续**基础战斗 + 技能 + 海克斯技能**三层。下一步要拍板首批角色数量、海克斯技能系统启动节点、大招是否通用、部位伤害倍率。
